- 150 samples

- Seaborn

## EDA Steps Performed
1. Data loading and inspection
2. Statistical summary
3. Histogram analysis
4. Categorical count plot
5. Outlier detection using box plots
6. Correlation heatmap
7. Pair plot visualization
8. Feature importance identification

---

## Key Insights
- Petal length and width are the most important features
- Dataset has no missing values
- No class imbalance
- Minimal outliers present
- Data is suitable for ML modeling
